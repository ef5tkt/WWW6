## 🌟 项目愿景与使命

### **我们的初衷：听见，即是疗愈**
你是否也曾有过这样的时刻？仅仅是听见一位姊妹轻声细语地分享，那种独特的声波频率就像一股清泉，瞬间抚平了内心的焦虑。我们发现，**女性的声音自带一种天生的磁场**——它温柔治愈，它坚定有力，它本身就是一种极其珍贵的能量。

### **我们的挑战：打破沉默的枷锁**
但在当下的主流叙事中，这些动人的声音在哪里？她们往往被喧嚣淹没，被偏见忽视，被算法过滤。**世界欠女性一份专注的倾听。** 这就是为什么我们需要 **Attention Please (听她说)**。

### **我们的使命**
我们建立这个空间，是为了让每一位姊妹的声音都能找到降落的土壤：
* **让声音流动**：打造一个纯净的声音档案库，让那些治愈的力量在每一个孤独的灵魂间流淌。
* **让发声永恒**：借助 Web3 技术的加持，让你的声音不再是随风而逝的震动，而是永远镌刻在链上的数字遗产。
* **让创造力回归**：夺回叙事的主权。在这里，你不是被定义的客体，你是用声音重塑世界的创作者。

## 💡 核心场景：在声波中重拾力量

基于 Vite + React 的极简交互，我们为你设计了两种深度连接的方式：

### 1. 疗愈时刻 (Healing Mode)
* **随机相遇**：打开网页，随机开启一段来自陌生姊妹的真实声音。
* **深度共鸣**：在她温柔的频率里，找到片刻的宁静。没有繁杂的交互，只有你与她之间跨越时空的听觉连接。

### 2. 她力量板块 (Power Mode)
* **理想与勇气**：这里汇聚了关于女性理想、价值观、成长阵痛与突破的故事。
* **能量补给**：当你感到低落、迷茫时，点开这个板块。那些坚定的声音会告诉你：你并不孤单，你的勇气已有先驱。

## 🛠️ Web3 赋能：让你的声音成为资产

我们拒绝传统平台对女性创作内容的免费剥削。在 **Attention Please**，你的每一分发声都有价值：

* **声音资产化 (NFT)**：你的故事、你的梦想、你的声音，都可以被铸造成独一无二的数字信物。它们通过 **Pinata & IPFS** 进行去中心化存储，永不丢失，真正归你所有。
* **安全门禁**：我们通过 **Wallet-gate (钱包门禁)** 与邀请制（如 `SISTER2024`）确保社区的纯粹性，只有真正的姊妹才能进入这个共建者空间。
* **价值回馈**：每一位分享声音、贡献内容的姊妹，都是平台的共建者。你的每一次发声，都在为社区创造价值，而这个价值会通过代币机制真实地回馈给你。

* [x] **Web3 验证体系**：完整实现了基于 MetaMask 连接与邀请码验证的 `WalletGateModalReal` 逻辑。

* [x] **沉浸式交互框架**：开发了支持全屏滑动、沉浸式播放的音频卡片框架。

### 下一步行动计划 🎯
* **动态声音广场**：接入后端 API，实时展示社区成员最新录制的真实声音。
* **录音确权工具**：在录音页面加入自动关联钱包地址的逻辑，实现“录制即确权”。
* **链上勋章体系**：为持续发声的姊妹颁发基于灵魂绑定代币（SBT）的成长勋章。
